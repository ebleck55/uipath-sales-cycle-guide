Testing Instructions

1. Test Hardened Version

  • • Click "Enter Edit Mode" to activate admin features
  • • Click "Admin Panel" to access analytics and AI settings
  • • Test the industry switcher (Banking/Insurance)
  • • Verify all content loads properly

2. Test AI Features

  • • Go to Admin Panel → AI Settings
  • • Enter your Claude API key
  • • Test the connection
  • • Check Analytics tab for prompt tracking

3. Test Analytics

  • • Make some AI requests
  • • Check the Analytics tab for tracked prompts
  • • Test data export functionality
  • • Verify prompt quality assessment

4. Test Admin Interface

  • • Access the standalone admin interface
  • • Test content management features
  • • Verify site statistics
  • • Check AI content recommendations

Browser Console Testing

Open browser dev tools and test these in the console:

// Test global objects
window.HardenedApp

await aiService.generateResponse('Hello, this is a test')
